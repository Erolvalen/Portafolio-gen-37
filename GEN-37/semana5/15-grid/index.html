Grid CSS

Conceptos básicos

El sistema de grid de CSS es un sistema de filas y columnas que nos permite organizar el contenido de una página web de una forma más ordenada y estructurada. Actualmente es el sistema más poderoso en CSS, ya que a diferencia de Flxbox, el cual es unidimensional, grid es un sistema bidimensional.

  • Contenedor: es el elemento que contiene a todos los elementos que queremos organizar.
  • Elementos: son los elementos que queremos organizar dentro del contenedor.
  • Filas: son las líneas horizontales que dividen el contenedor.
  • Columnas: son las líneas verticales que dividen el contenedor.
  • Celdas: son las intersecciones de las filas y columnas.
  • Áreas: son las áreas que se forman al unir varias celdas.
  • Líneas: son las líneas que dividen las filas y columnas.
  • Bandas: son las áreas que se forman entre dos líneas consecutivas.

Propiedades del contenedor

Para poder utilizar el sistema de grid de CSS, primero debemos definir un contenedor que será el que contendrá a todos los elementos que queremos organizar. Para ello, utilizamos la propiedad display con el valor grid.

Grid explícito

El grid explícito es el grid que se crea cuando se especifica el número de filas y columnas.

Una vez definido el contenedor, podemos definir las filas y columnas que queremos que tenga. Para ello, utilizamos la propiedad grid-template-columns y grid-template-rows. En estas propiedades, podemos definir el tamaño de las filas y columnas utilizando unidades de medida como px, em, rem, %, etc.

Grid Areas

Mediante los grids CSS es posible indicar el nombre y posición concreta de cada área de una cuadrícula. Para ello utilizaremos la propiedad grid-template-areas, donde debemos especificar el orden de las áreas en la cuadrícula. Posteriormente, en cada ítem hijo, utilizamos la propiedad grid-area para indicar el nombre del área del que se trata:
